# Case Transfers

This directory holds per-case transfer memos written by `/legal-aid:case-transfer`.

Layout:

```
case-transfers/
└── [YYYY-MM]/
    ├── _summary.md            # one-line entry per transfer this month
    └── [case-id].md           # full transfer memo
```

The plugin ships this directory empty. Memos accumulate as transfers happen. Treat the files as the operational record — missed transfers are a malpractice exposure point.
