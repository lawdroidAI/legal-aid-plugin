# Client Communications Log

This directory holds per-case communication logs written by `/legal-aid:client-comms-log`.

Layout:

```
client-comms/
└── [case-id]/
    └── log.md                 # append-only log of calls, emails, letters, in-person
```

Append-only. Substantive legal analysis does not live here — that lives in memos and status files. This is the record of who said what to whom and when.
